# 【Python学习推荐】
人生苦短，当学Python。
这里给出一些，上手比较快的好用的Python学习建议，希望大家能按时完成。

[北理工Python入门课程](https://www.icourse163.org/course/BIT-268001) 在MOOC平台上，北理工的Python课程比较完整，是套全家桶，后续网络爬虫、数据分析以及机器学习，都有相应的课程。
超链接给出的是最入门的课程，但是视频看起来比较慢，并不方便迅速掌握和实践。
